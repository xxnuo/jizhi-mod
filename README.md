## 介绍

支持自定义新标签页的跨浏览器扩展，浮生梦将在新标签页上展示中国经典诗词。

### 目前已完成的功能

- 点击诗句自动朗读
- 具有多种风格迥异的字体
- 支持自动切换明暗主题

有空会持续更新新功能，目前只是基础 1.0 版本。

## 特性

- 使用完全离线的数据，无需联网
- 保持设计风格：简洁美观第一，速度性能第一（针对低功耗设备优化）
- 使用 WXT + React + Tailwindcss + Daisyui 开发
